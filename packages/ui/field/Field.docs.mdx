# Field

라벨, 입력, 설명, 에러 메시지를 묶는 레이아웃 컴포넌트입니다.

## 기본 조합

`Field` 안에 `FieldLabel`과 입력 컴포넌트를 넣으면 됩니다.

```tsx
<Field>
  <FieldLabel htmlFor="name">이름</FieldLabel>
  <TextInput />
</Field>
```

<Canvas of={FieldStories.WithLabelAndInput} />

## 설명 추가

입력 힌트가 필요하면 `FieldDescription`을 추가합니다.

```tsx
<Field>
  <FieldLabel htmlFor="email">이메일</FieldLabel>
  <TextInput />
  <FieldDescription>사용 가능한 이메일 주소를 입력하세요.</FieldDescription>
</Field>
```

<Canvas of={FieldStories.WithDescription} />

## 에러 상태

유효성 검사 실패 시 `FieldError`를 추가합니다. `errors` prop에 배열을 넘기면 중복 제거 후 렌더됩니다.

```tsx
<Field>
  <FieldLabel htmlFor="password">비밀번호</FieldLabel>
  <TextInput />
  <FieldError errors={[{ message: '비밀번호는 8자 이상이어야 합니다.' }]} />
</Field>
```
